Back-end API

Node.js, RESTful

About

This API is build for storing data of animals. You add their name, age, diet and which animal they are.
Creating this gave me insight in the workings of API's and taught me about HTTP requests.
The API also has a pagination for ease of use.

Technology

This backend API was made using Node.js, Express and Mongoose.
It only accepts strings as input because that was a requirement from school, which was a bit weird, but sure.
