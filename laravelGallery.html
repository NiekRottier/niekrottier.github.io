Laravel Gallery

Laravel, MVC, MySQL

About

With this web application you can login and look at and post pictures with a title, description and tag.
Other functions include liking pictures, hiding, editing and deleting your own pictures, and searching based on the title or tag.
At the moment there are 5 tags, but this could be expanded.

Technology

The Laravel Gallery is, as the name suggests, build with the PHP framework Laravel.
Laravel works with the MVC pattern, so this application also makes use of this.
The application has a MySQL database.
